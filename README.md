### Install

<https://packagecontrol.io/packages/Hexa>

#### Dependencies

- [SublimeLinter](http://www.sublimelinter.com/en/stable/installation.html)

#### Supported features

- Syntax highlight
- Syntax errors (stand-alone & with compiler)
- Build systems
- Code navigation (basic)
- Snippets
